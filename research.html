UnderGraduate Research Work(Thesis)


  1. Prediction of COVID-19 from Chest X-ray Images Using CNN And Pre-Trained Models
    • COVID-19 infected patients show abnormalities in chest X-ray images. However,
    examining the chest X-rays requires a specialist with high experience. Hence, using deep
    learning techniques in detecting abnormalities in the X-ray images is presented commonly
    as a potential solution to help diagnose the disease. Numerous researchers have already
    been approached various techniques on the COVID-19 chest X-ray images. Here, I have
    trained seven pre-trained models on the COVID-19 chest X-ray dataset. Seven pre-trained
    models are:

    • Visual Geometry Group(VGG16)
    • Residual Neural network (ResNet50)
    • Inception Version 3 (InceptionV3)
    • Efficient Network (EfficientNetB0)
    • Neural Architecture Search Network Mobile (NasNetMobile)
    • Visual Geometry Group (VGG19)
    • Dense Network (DenseNet201)



  2. Analysis
    • We got the best accuracy result from VGG16.

    • VGG16 accuracy is 0.9767

    • and we got the lowest accuracy from the

    • Pre-trained CNN model EfficientNetB0.

    • Lowest accuracy of EfficientNetB0 is 0.5000.
